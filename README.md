## Usage

1. Clone this repo

1. Install dependancies

   ```
   yarn
   ```

1. Review outdated dependancies and upgrade if needed

   ```
   yarn outdated
   ```

1. Run app

   ```
   yarn dev
   ```

   or if you have vercel-cli installed,

   ```
   vercel dev
   ```
